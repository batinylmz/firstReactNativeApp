# My First React Native Project 🚀

Welcome to my very first React Native application! This project marks the beginning of my journey as a mobile software developer. I built this app to practice the core fundamentals and get comfortable with the React ecosystem.

---

## 🛠 Key Features

*   **Authentication Flow**: Realistic Login and Register screens with validation logic.
*   **Navigation**: Seamless transition between screens using React Navigation.
*   **State Management**: Utilizing `Context API` for global data handling.
*   **Modern UI**: Clean and responsive interfaces designed with Figma.
*   **Dynamic Theme**: Dark and Light mode support using `ThemeContext`.

---

## 🚀 Tech Stack

| Technology | Usage |
| :--- | :--- |
| **React Native** | Core Framework |
| **StyleSheet** | UI Styling (CSS-in-JS) |
| **Figma** | UI/UX Design |
| **WebStorm** | Primary IDE |

## 📁 Project Structure

```text
src/
 ├── context/    # Global state management (Auth, Theme)
 ├── screens/    # Main application screens (Login, Profile, etc.)
 ├── components/ # Reusable UI elements
```
